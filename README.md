# Agent Skills

Agent Skills are reusable capabilities for AI agents. Install them with a single command to enhance your agents with access to procedural knowledge.

> Read about on [agentskills.io](https://agentskills.io)

## Getting Started

To install a skill, use the `skills` CLI:

```sh
npx skills add https://github.com/gustavo-flor/agent-skills --skill <skill-name>
```
